# Проект: Путишествия по России

**Обзор**

Проект создан для того чтобы показыть красивые места в России которые можно посетить
### Использованные технологии
*БЭМ
*Flex-box
*Адаптивная верстка (под популярные разрешения)
*Резтновость сайта
*Grid разметка
